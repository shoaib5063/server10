# Car Rental Platform - Backend API

A RESTful API for car rental services built with Node.js, Express, and MongoDB. Features Firebase authentication, complete CRUD operations for cars and bookings, and comprehensive error handling. Deployed on Vercel with MongoDB Atlas integration.

**Live API:** https://server10-mu.vercel.app

## Tech Stack
Node.js, Express, MongoDB, Mongoose, Firebase Admin SDK, CORS

## Key Features
- Car listing management (create, read, update, delete)
- Booking system with availability tracking
- Firebase authentication for secure endpoints
- Provider-specific car filtering
- User booking history
- Database seeding for sample data

## Quick Start
```bash
npm install
npm run dev
```

Configure `.env` with MongoDB URI and Firebase credentials. API runs on port 5000.
